# David Ayers Dotfiles & Misc Setup Stuff

## Installing dotfiles

To install your dotfiles onto a new system:

    $ cd $HOME
    $ git clone git@github.com:username/dotfiles.git .dotfiles
    $ ./.dotfiles/bin/dfm install # creates symlinks to install files
